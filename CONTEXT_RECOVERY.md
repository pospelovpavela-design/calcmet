## Критические открытые вопросы

### ❗ Вопрос 1 (КРИТИЧЕСКИЙ): Формула подкрановых балок М1

**Проблема:** Код в `main_desktop.py:527` (`_pb_row`) использует формулу, идентичную расчёту
собственного веса подкрановой балки `Gpb` из расчёта колонны (строка 565):

```python
# КОД (строка ~527):
Gпб = αпб * Lпб * (qэкв_нагрузка * Lпб + Gрельс) * 1.4 / 9.81 * mf
# На самом деле: (αпб×Lпб + Gрельс)×Lпб×1.4/9.81×mf  ← CRANE_Q_EQUIV не используется
```

```
# МЕТОДИКА (разд. 5.1):
Gпб = αпб × Lпб × (qэкв × Lпб + Gрельс)
```

**Анализ:**
- `CRANE_Q_EQUIV` полностью отсутствует в `_pb_row`
- Методическая формула нарушает размерность: qэкв [кН/м] × Lпб [м] = [кН], а Gрельс [кН/м] → нельзя складывать
- Отношение кода к плану: 13.9× при 5т, 0.83× при 200т — формулы вычисляют разные вещи
- **Гипотеза:** формула скопирована из расчёта колонны и не заменена правильной методической формулой

**Что нужно:** Найти оригинальный источник методики ЦНИИ, разд. 5.1, уточнить единицы αпб и qэкв.
