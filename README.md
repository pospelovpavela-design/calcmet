# MetalCalc Suite — Расчёт металлоёмкости производственных зданий

Десктопное приложение для определения металлоёмкости одноэтажных производственных зданий с мостовыми кранами, трубопроводных и электрокабельных эстакад.

## Возможности

### Производственные здания (`main_desktop.py`)

- **Многопролётные здания** (до 6 пролётов) с независимыми параметрами каждого пролёта
- **Два метода расчёта:**
  - **Метод 1** — аналитические формулы (полностью встроен, внешние файлы не нужны)
  - **Метод 2** — табличные данные из xlsx/docx файлов (если файлы не найдены — используются встроенные значения)
- **Кровельный пирог** — конструктор из ~30 слоёв (утеплители, настилы, гидро- и пароизоляция) с авторасчётом нагрузки кН/м² и 4 предустановленными типами
- **Режим работы крана:** 1К–6К (лёгкий/средний) и 7К–8К (тяжёлый) с корректными коэффициентами для каждого метода
- **Прогоны:** шаг ферм B=6 м (швеллеры) и B=12 м (двутавры ГОСТ 26020) по Таблице 3 методики
- **Экспорт результатов** в текстовый файл
- **Всплывающие подсказки** (`?`) по каждому параметру со ссылками на нормативы

### Трубопроводные эстакады (`estakada_pipe.py`, v3.2F)

- 9 типоразмеров от 1 до 9 трубопроводов
- Ввод длины эстакады; вывод: масса металлоконструкций (т), масса на м.п.

### Электрокабельные эстакады и галереи (`estakada_elec.py`, v4.2F)

- Эстакады без прохода и галереи с проходом
- Несколько типоразмеров по ширине и нагрузке

## Установка и запуск

```bash
pip install -r requirements.txt
python launcher.py
```

**Зависимости:** `customtkinter >= 5.2`, `pandas >= 2.0`, `openpyxl >= 3.1`, `python-docx >= 1.1`

### Файлы таблиц (Метод 2)

Программа ищет файлы в следующем порядке:
1. Папка рядом с `main_desktop.py`
2. `~/Desktop/Металлоемкость/Тип 1 здания с кранами/`

| Файл | Содержимое |
|---|---|
| `металлоекмсоть покрытия.xlsx` | Фермы (Уголки/Двутавры/Молодечно), подстропильные, связи |
| `Металлоёмкость фахверк.xlsx` | Фахверк по схемам I/II/III, опоры трубопроводов |
| `Таблица металлоемкости на подкрановые конструкции.docx` | Подкрановые балки по г/п и шагу |
| `Таблица металлоемкости на тормозные конструкции.docx` | Тормозные конструкции |

Если файлы не найдены — программа работает полностью в режиме Метод 1.

## Кровельный пирог

Вместо ввода суммарной нагрузки числом, пользователь собирает пирог из слоёв:

**Предустановленные конфигурации:**
- Унипрофиль (профнастил + минвата 150 мм + ПВХ мембрана)
- Сэндвич-панель 200 мм (готовая кровля)
- Утеплённая PIR 150 мм + наплавляемая гидроизоляция
- Холодная кровля (профнастил без утеплителя)

**Доступные слои (~30 материалов):**

| Группа | Примеры |
|---|---|
| Несущий настил | Профнастил Н-75/Н-60/НС-35 (t=0.5–0.8 мм) |
| Утеплитель минвата | 50/100/150/200 мм (ρ=80 кг/м³) |
| Утеплитель PIR | 80/100/150/200 мм (ρ=35 кг/м³) |
| Утеплитель пенополистирол | 100/150/200 мм (ρ=25 кг/м³) |
| Пароизоляция | Полиэтиленовая плёнка, фольгированная, битумная |
| Гидроизоляция | ПВХ/ТПО/ЭПДМ мембрана 1.5 мм, наплавляемая СБС 2 слоя |
| Готовые системы | Сэндвич-панель кровельная 150/200 мм |
| Прочие | Стяжка цементная, доборные элементы |

Нагрузку можно также ввести вручную, установив соответствующий флажок.

## Методика расчёта

### Прогоны

Нагрузка на прогон:
```
q = (Q_кровля + g_св·γf + Q_снег·γf_с + Q_пыль·γf_п + 1.0·γc_т) · a_пр · γc
```
Шаг прогонов `a_пр = 3.0 м`. Профиль подбирается по **Таблице 3 методики**:

| B | Профиль |
|---|---|
| 6 м | Швеллеры ГОСТ 8240 (Ш20–Ш30, 2×Ш20) |
| 12 м | Двутавры Б-серии ГОСТ 26020 (30Б1–2×30Б1) |

### Колонны

Высоты рассчитываются с учётом высоты подкрановой балки и рельса:

- **Надкрановая часть:** `H_v = H₂ − h_рельса + h_б + h_р`
- **Подкрановая часть:** `H_n = h_рельса − h_б − h_р + H_F`

Высота рельса по г/п крана:

| Г/П, т | Тип рельса | h_р, м |
|---|---|---|
| ≤ 20 | КР70 | 0.130 |
| ≤ 50 | КР80 | 0.150 |
| ≤ 80 | КР100 | 0.170 |
| > 80 | КР120 | 0.180 |

### Подкрановые балки и режим работы крана

Применяются **раздельные коэффициенты** для каждого метода:

| Метод | База таблиц | Коэф. 1К–6К | Коэф. 7К–8К |
|---|---|---|---|
| М1 (формула) | 1К–6К | 1.00 | **×1.80** |
| М2 (таблицы) | 7К–8К | ×0.65 (−64%) | **1.80** |

Коэффициент 7К–8К = **1.80** откалиброван по методике (разд. 5.2). Снижение М2 для режима 1К–6К составляет ~64% от режима 7К–8К.

### Расчётные элементы

| Конструкция | Метод 1 | Метод 2 |
|---|---|---|
| Прогоны | ✓ | — |
| Стропильные фермы (Уголки) | ✓ | — |
| Стропильные фермы (Двутавры, Молодечно) | — | ✓ |
| Связи по покрытию | — | ✓ |
| Подстропильные фермы | ✓ | ✓ |
| Подкрановые балки + тормозные | ✓ | ✓ |
| Колонны (подкрановая и надкрановая части) | ✓ | — |
| Фахверк | — | ✓ |
| Опоры трубопроводов | — | ✓ |

## Эстакады

### Трубопроводные (`estakada_pipe.py`, v3.2F)

9 типоразмеров (1–9 трубопроводов, Ø до 500 мм). Ключевые поля CONFIGS:

| Поле | Описание |
|---|---|
| `num` | Номер типоразмера (уникальный) |
| `max_pipes` | Максимальное количество труб |
| `max_d_mm` | Максимальный диаметр трубы, мм |
| `std_load` | Стандартная нагрузка, т/м |
| `svc_load` | Обслуживающая нагрузка, т/м |
| `kg_m` | Удельная масса конструкции, т/м |
| `pipes_low/high` | Диапазон числа труб |

### Электрокабельные и галереи (`estakada_elec.py`, v4.2F)

Два типа: `type_id=0` — эстакада без прохода, `type_id=1` — галерея с проходом. Галереи тяжелее эстакад.

## Структура данных `calculate()`

Функция `calculate(gp, spans)` принимает:

```python
gp = {
    "L_build": float,   # длина здания, м
}
spans = [
    {
        "L":         float,   # пролёт, м
        "B_step":    float,   # шаг ферм, м (6 или 12)
        "col_step":  float,   # шаг колонн, м (6 или 12)
        "truss_type": str,    # "Уголки" / "Двутавры" / "Молодечно"
        "q_crane_t": float,   # г/п крана, т
        "n_cranes":  int,     # 1 или 2
        "with_passage": bool, # тормозные с проходом
        "crane_mode": str,    # "Режим 1-6К" / "Режим 7-8К"
        "h_rail":    float,   # УГР, м
        "H_col_ov":  float,   # высота колонны (0 = авто)
        "Q_snow":    float,   # снег, кН/м²
        "Q_dust":    float,   # пыль, кН/м²
        "Q_roof":    float,   # кровля, кН/м² (из кровельного пирога)
        "yc":        float,   # γc
        "fakhwerk_load": float,
        "fakhwerk_post": bool,
        "building_type": str,
    },
    # ... следующие пролёты
]
```

Возвращает список словарей с ключами `"m1"`, `"m2"`, `"purlin"`, `"log"` для каждого пролёта.
