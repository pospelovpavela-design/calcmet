# Расчёт металлоёмкости производственных зданий

Десктопное приложение для определения металлоёмкости одноэтажных производственных зданий с мостовыми кранами.

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![CustomTkinter](https://img.shields.io/badge/UI-CustomTkinter-informational)
![License](https://img.shields.io/badge/license-MIT-green)

## Возможности

- Многопролётные здания (до 6 пролётов) с **независимыми параметрами** каждого пролёта
- Два метода расчёта: эмпирические формулы (Метод 1) и табличные данные xlsx/docx (Метод 2)
- Учёт режима работы крана (1К–6К и 7К–8К)
- Корректное разделение на подкрановую и надкрановую части колонны
- Всплывающие подсказки (`?`) по каждому параметру со ссылками на нормативы
- Экспорт результатов в текстовый файл

## Файлы таблиц (Метод 2)

Программа ищет файлы в следующем порядке:
1. Папка рядом с `main.py`
2. `~/Desktop/Металлоемкость/Тип 1 здания с кранами/`

| Файл | Содержимое |
|---|---|
| `металлоекмсоть покрытия.xlsx` | Фермы (Уголки/Двутавры/Молодечно), подстропильные, связи |
| `Металлоёмкость фахверк.xlsx` | Фахверк по схемам I/II/III, опоры трубопроводов |
| `Таблица металлоемкости на подкрановые конструкции.docx` | Подкрановые балки по г/п и шагу |
| `Таблица металлоемкости на тормозные конструкции.docx` | Тормозные конструкции |

Если файлы не найдены — программа использует встроенные значения по умолчанию (Метод 1 или константы).

## Методика расчёта

### Прогоны (Метод 1)
Нагрузка на прогон: `q = (Q_кровля + g_св·γf + Q_снег·γf_с + Q_пыль·γf_п + 1.0·γc_т) · a_пр · γc`
Шаг прогонов `a_пр = 3.0 м`. Масса подбирается по таблице (PROGON_6M / PROGON_12M) в зависимости от шага ферм.

### Колонны (Метод 1)
Высоты рассчитываются с учётом высоты подкрановой балки (h_б) и рельса (h_р):
- **Надкрановая часть:** `H_v = H₂ − h_рельса + h_б + h_р`
- **Подкрановая часть:** `H_n = h_рельса − h_б − h_р + H_F`

Где `H₂ = h_рельса + H_кран + 0.1 + A_зазор` — отметка головы колонны; `H_F = 0.6 м` — заглубление базы.

Высота рельса по г/п крана (Таблица 4 методики):

| Г/П, т | Тип рельса | h_р, м |
|---|---|---|
| ≤ 20 | КР70 | 0.130 |
| ≤ 50 | КР80 | 0.150 |
| ≤ 80 | КР100 | 0.170 |
| > 80 | КР120 | 0.180 |

### Подкрановые балки (Методы 1 и 2)
Коэффициент режима: `αпб` из таблицы по г/п и режиму крана.
Для Метода 2 применяется `mode_ratio = (αпб·B + q_р) / (αпб₇₈·B + q_р)` — поправка на режим 1К–6К.
