# Changelog

## [1.1.0] — 2026-02-28
### Добавлено
- Desktop-версия на CustomTkinter (Windows / macOS)
- GitHub Actions: автосборка EXE и DMG
- Многопролётные здания с гибридным суммированием
- Коррекция высоты колонн
- Кровельный пирог (несколько вариантов)
- Режимы кранов (лёгкий / средний / тяжёлый)

## [1.0.0] — 2026-02-21
### Добавлено
- Android-версия на Kivy / KivyMD
- Расчёт ферм (уголки, двутавры, молодечненский прокат)
- Подкрановые балки и крановые рельсы
- Тормозные конструкции
- Фахверк (торцевой и продольный)
- Прогоны покрытия
- GitHub Actions: автосборка APK
